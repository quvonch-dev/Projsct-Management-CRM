# Project Management CRM Dashboard

Bu loyiha Figma CRM templatega asoslangan to'liq funksional dashboard hisoblanadi.

## Fayllar
- `index.html` — Asosiy dashboard fayl (barcha HTML, CSS va JS bitta faylda)

## Xususiyatlar
- Qorong'i sidebar navigatsiya
- 4 ta statistika kartasi (Completed, Pending, Income, Expense)
- Quick Action tugmalari
- Loyihalar jadvali (filtrlash imkoniyati bilan)
- Donut chart (Awards)
- Bar chart (Overall Summary — Yearly/Monthly)
- Crypto assets paneli
- Responsive dizayn (mobil va planshet uchun)

## Ishlatish
`index.html` faylini brauzerda oching — hech qanday server yoki o'rnatish talab qilinmaydi.

## Texnologiyalar
- Sof HTML, CSS, JavaScript
- Tabler Icons (CDN orqali)
- Hech qanday freymvork yoki kutubxona kerak emas
